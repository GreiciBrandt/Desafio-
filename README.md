# 💻 Portfólio Pessoal – Greici Brandt

Este projeto é um **site de portfólio pessoal** desenvolvido para apresentar minhas habilidades, projetos e formas de contato como **Desenvolvedora Web**.

O objetivo do portfólio é reunir meus principais trabalhos e demonstrar minha evolução na área de desenvolvimento web.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido como parte dos estudos no curso **SCTech**, com o objetivo de praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Interatividade com JavaScript
* Organização de projetos para portfólio

---

## 📌 Funcionalidades

* Apresentação pessoal
* Seção "Sobre mim"
* Lista de projetos desenvolvidos
* Tecnologias utilizadas em cada projeto
* Links para repositórios no GitHub
* Seção de contato com e-mail e telefone
* Layout simples e responsivo

---

## 🛠️ Tecnologias utilizadas

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout
* **JavaScript** – Interatividade do site

---

## 🚀 Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/GreiciBrandt/Desafio-.git
```

2. Acesse a pasta do projeto:

```
cd Desafio-
```

3. Abra o arquivo **index.html** no navegador.
